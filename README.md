# 🍗 Monopoly Chicken

A modern restaurant landing page designed for a fictional chicken restaurant called **Monopoly Chicken**.

The website focuses on a clean user interface, smooth user experience, and responsive design that works well on both desktop and mobile devices.

---

## 🌐 Live Demo

You can view the live website here:

👉  https://monopoly-chicken-zoubir.vercel.app/



---


## ✨ Features

- Modern restaurant landing page
- Responsive design (mobile-friendly)
- Smooth scrolling sections
- Reservation form interface
- Clean UI and simple navigation
